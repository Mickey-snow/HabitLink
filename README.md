# HabitLink

## Obtain the Code:

```bash
git clone https://github.com/PJ2-group2/HabitLink.git
cd HabitLink/
```

## Run the Server:

```bash
mvn exec:java -Dexec.mainClass=com.habit.server.HabitServer
```

## Run the Clients:

```bash
mvn exec:java -Dexec.mainClass=com.habit.client.gui.HabitClientGUI
```


## Run the Tests:

```bash
mvn test
```
